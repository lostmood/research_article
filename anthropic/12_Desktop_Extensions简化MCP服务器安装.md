# Desktop Extensions: 简化 MCP 服务器安装

## 文章信息
- **标题**: Desktop Extensions: One-Click MCP Server Installation
- **发布日期**: 2025-06-26
- **链接**: https://www.anthropic.com/engineering/desktop-extensions

## 核心内容总结

这篇文章介绍了 Anthropic 推出的 **Desktop Extensions (.mcpb 文件)**,一种将 MCP 服务器与所有依赖项捆绑到**单个可安装归档**的打包格式,极大简化了 MCP 服务器的安装流程。

### 核心功能

**打包格式**:
- **.mcpb 文件**: 单个可安装的归档文件
- 包含 MCP 服务器和所有依赖项
- 一键安装,无需手动配置

**系统特性**:
1. **运行时管理**: 自动管理运行时环境
2. **自动更新**: 自动检查和安装更新
3. **安全凭据存储**: 通过操作系统密钥链存储凭据
4. **消除手动配置**: 无需手动编辑配置文件

### 开发支持

**支持的技术栈**:
- Node.js
- Python
- 二进制可执行文件

**配置文件**:
- **manifest.json**: 声明功能、用户配置需求和平台特定设置
- 定义扩展的元数据和依赖
- 指定权限和资源需求

### 用户体验

**安装流程**:
1. 下载 .mcpb 文件
2. 双击安装
3. 配置必要的参数(如 API 密钥)
4. 开始使用

**维护**:
- 自动检查更新
- 一键更新到新版本
- 统一的管理界面

## Desktop Extensions 架构

```
.mcpb 文件
├── manifest.json (元数据和配置)
├── 运行时环境 (Node.js/Python/Binary)
├── MCP 服务器代码
├── 依赖项
└── 资源文件
```

## manifest.json 示例

```json
{
  "name": "example-mcp-server",
  "version": "1.0.0",
  "description": "An example MCP server",
  "runtime": "node",
  "entry": "index.js",
  "permissions": [
    "filesystem:read",
    "network:http"
  ],
  "config": {
    "api_key": {
      "type": "string",
      "description": "API key for the service",
      "required": true,
      "secure": true
    }
  },
  "platforms": {
    "macos": {
      "min_version": "10.15"
    },
    "windows": {
      "min_version": "10"
    },
    "linux": {
      "min_version": "Ubuntu 20.04"
    }
  }
}
```

## 安装流程

**用户视角**:
1. 下载 .mcpb 文件
2. 双击文件
3. 系统提示安装
4. 输入必要的配置(如 API 密钥)
5. 安装完成,工具可用

**系统视角**:
1. 解析 manifest.json
2. 检查平台兼容性
3. 安装运行时环境
4. 解压 MCP 服务器和依赖
5. 配置凭据存储
6. 注册到系统
7. 启动 MCP 服务器

## 更新机制

**自动更新流程**:
1. 定期检查更新
2. 发现新版本
3. 下载更新包
4. 验证签名
5. 安装更新
6. 重启 MCP 服务器
7. 通知用户

**版本管理**:
- 语义化版本控制
- 向后兼容性检查
- 回滚机制
- 更新日志

**权限控制**:
- 声明式权限模型
- 最小权限原则

**代码签名**:

- 建立信任链
