# Introducing o3 and o4-mini

## 基本信息
- **来源**: OpenAI Blog
- **发布日期**: 2025-04-16
- **链接**: https://openai.com/index/introducing-o3-and-o4-mini/
- **类别**: 模型发布 / 推理模型 (Reasoning Models)

## 核心概述

OpenAI 发布了两款新一代推理模型 **o3** 和 **o4-mini**，被称为"迄今为止发布的最智能模型"。o3 是全能型旗舰推理模型，在编码、数学、科学和视觉推理任务上表现最强；o4-mini 则是面向速度和成本效率优化的紧凑版本，在数学推理上甚至超越 o3。两款模型首次实现了推理模型的**原生智能体工具调用** (Agentic Tool Use)，能够在推理过程中自主决定何时、如何调用工具，标志着从"思考型模型"向"行动型智能体"的关键演进。

## 关键技术细节

### 模型规格

| 特性 | o3 | o4-mini |
|------|-----|---------|
| **定位** | 旗舰推理模型 | 高效推理模型 |
| **上下文窗口 (Context Window)** | 200K tokens | 200K tokens |
| **最大输出 (Max Output)** | 100K tokens | 100K tokens |
| **知识截止 (Knowledge Cutoff)** | 2024-05-31 | 2024-05-31 |
| **输入定价** | $2.00 / 1M tokens | $1.10 / 1M tokens |
| **输出定价** | $8.00 / 1M tokens | $4.40 / 1M tokens |
| **多模态输入** | 文本 + 图像 | 文本 + 图像 |

### 基准测试表现

| 基准测试 | o3 | o4-mini | Gemini 2.5 Pro |
|----------|-----|---------|----------------|
| **AIME 2024** (数学竞赛) | 91.6% | **93.4%** | 92.0% |
| **AIME 2025** (数学竞赛) | 88.9% | **92.7%** | 86.7% |
| **SWE-Bench** (代码) | **领先** | 强劲但有波动 | - |

- o4-mini 在数学基准测试上全面超越 o3 和 Gemini 2.5 Pro
- o3 在复杂推理和代码任务（SWE-Bench）上保持领先
- 两者均支持 "high" 推理模式 (o4-mini-high) 以提升可靠性

### 原生智能体工具调用 (Agentic Tool Use)

这是 o3/o4-mini 最核心的突破。两款模型首次能够在推理链 (Chain-of-Thought) 中**原生、自主地调用所有 ChatGPT 工具**：

- **Python 代码执行** (Code Execution) — 在推理过程中编写和运行代码
- **Web 浏览** (Web Browsing) — 实时搜索和获取网页信息
- **图像理解与分析** (Image Understanding) — 理解上传的图片、白板、草图
- **图像生成** (Image Generation) — 调用图像生成工具
- **Canvas 创建** — 在画布上创作内容

模型可以自主**链式组合多个工具**完成复杂工作流：如浏览网页获取数据 -> 编写 Python 分析 -> 生成图表可视化结果，全程无需人工干预。

### "用图像思考" (Thinking with Images)

o3 和 o4-mini 是 OpenAI 首批能够**将视觉信息直接整合到推理链**中的模型：
- 不仅"看到"图像，还能在思维链中主动**裁剪图像、分析图像细节**
- 能理解低质量的白板照片、手绘草图和图表
- 工具在思维链中的使用带来了"显著的改进" (significant improvements)

### 可用性

- **ChatGPT**: Plus、Pro、Team、Enterprise 用户可直接使用
- **API**: 通过 Chat Completions API 和 Responses API 开放
- Free 用户可有限使用

## 核心洞察

1. **推理模型的"行动化"转折点**：o3/o4-mini 标志着推理模型从"只能思考"进化到"能思考也能行动"。原生工具调用使推理链不再是封闭的文本推理过程，而是可以与外部世界交互的开放系统。这是迈向通用智能体的关键一步。

2. **小模型的逆袭**：o4-mini 在数学推理上全面超越 o3，挑战了"越大越好"的直觉。这表明**针对特定能力维度的强化学习优化**可以让小模型在特定领域超越大模型，对成本敏感的部署场景意义重大。

3. **推理模型家族的终章与 GPT-5 的序章**：Sam Altman 暗示 o3/o4-mini 是最后的独立推理模型系列，GPT-5 将实现"传统能力 + 推理能力"的大一统。这意味着当前的 o-系列和 GPT-系列的双轨路线即将合并，o3 的所有能力将被 GPT-5 吸收。

4. **200K 上下文 + 100K 输出的意义**：超长输出能力使模型能够生成完整的代码库、长篇分析报告，结合智能体工具调用，构成了端到端自动化工作流的基础设施。
